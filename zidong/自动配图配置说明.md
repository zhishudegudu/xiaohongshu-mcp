# 自动配图功能配置说明

## 功能介绍

自动配图功能可以根据 AI 生成的内容，自动生成图片搜索关键词，然后从 Pixabay 免费图库下载相关图片。

## 工作流程

```
1. AI 生成内容（标题 + 正文）
   ↓
2. AI 分析内容，生成英文关键词
   例如：["coffee", "cafe", "latte"]
   ↓
3. 从 Pixabay 搜索并下载图片
   ↓
4. 自动发布到小红书
```

## 配置步骤

### 步骤 1: 注册 Pixabay API Key（免费）

1. 访问：https://pixabay.com/api/docs/
2. 点击 "Get Started" 或 "Sign Up"
3. 注册账号（可以用 Google 账号快速注册）
4. 注册后会自动获得 API Key
5. 复制你的 API Key（类似：`12345678-abc123def456...`）

### 步骤 2: 配置 API Key

**方法 1: 使用环境变量（推荐）**

```powershell
# PowerShell
$env:PIXABAY_API_KEY="你的API_Key"

# 验证
echo $env:PIXABAY_API_KEY

# 运行
python example_auto_images.py
```

**方法 2: 修改 config.py**

```python
# config.py
pixabay_api_key: str = os.getenv("PIXABAY_API_KEY", "你的API_Key")
```

⚠️ **不推荐**：不要把 API Key 硬编码在代码中！

### 步骤 3: 运行自动配图示例

```bash
python example_auto_images.py
```

## 使用示例

### 示例 1: 完整自动化

```python
from config import config
from mcp_client import MCPClient
from ai_generator import AIGenerator
from publisher import Publisher

# 初始化
mcp_client = MCPClient(config.mcp_server_url)
ai_generator = AIGenerator(
    api_key=config.gemini_api_key,
    base_url=config.gemini_base_url,
    model=config.gemini_model,
    proxy=config.gemini_proxy
)
publisher = Publisher(
    mcp_client, 
    config.image_folder, 
    config.local_image_folder,
    pixabay_api_key=config.pixabay_api_key  # 传入 API Key
)

# 1. 生成内容
result = ai_generator.generate_from_scratch("上海咖啡店探店")

# 2. 生成关键词
keywords = ai_generator.generate_image_keywords(
    result["title"], 
    result["content"], 
    count=3
)

# 3. 自动下载配图
images = publisher.download_images_by_keywords(keywords, count=3)

# 4. 发布
publisher.publish(
    title=result["title"],
    content=result["content"],
    images=images
)
```

### 示例 2: 手动指定关键词

```python
# 不使用 AI 生成关键词，手动指定
keywords = ["coffee", "cafe interior", "latte art"]
images = publisher.download_images_by_keywords(keywords, count=3)
```

## 常见问题

### Q1: 没有配置 API Key 会怎样？

程序会使用本地图片文件夹中的图片，不会自动下载。

### Q2: API Key 有使用限制吗？

Pixabay 免费账号限制：
- **每小时 100 次请求**
- **每天 5000 次请求**

对于个人使用完全够用！

### Q3: 下载的图片保存在哪里？

保存在 `local_image_folder` 配置的目录：
- Docker 模式：`E:\docker-xiaohongshu\images`
- 本地模式：根据配置

文件名格式：`auto_关键词_随机数.jpg`

### Q4: 下载失败怎么办？

程序会自动回退到使用本地图片：

```python
images = publisher.download_images_by_keywords(keywords, count=3)

if not images:
    print("⚠️ 图片下载失败，使用本地图片")
    images = publisher.get_random_images(count=3)
```

### Q5: 可以用其他图库吗？

可以！修改 `image_downloader.py`，添加其他图库的 API：
- **Pexels**: https://www.pexels.com/api/
- **Unsplash**: https://unsplash.com/developers
- **Lorem Picsum**: https://picsum.photos/

### Q6: 关键词是中文还是英文？

AI 会自动生成**英文关键词**，因为 Pixabay 主要是英文搜索。

示例：
- 输入：上海咖啡店
- 关键词：coffee shop, cafe interior, latte art

### Q7: 图片质量如何？

Pixabay 提供高质量的免费图片：
- 专业摄影师作品
- 无版权限制
- 可商用
