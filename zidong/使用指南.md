## 📝 修改主题

### 方法 1: 修改示例文件

编辑 `example_auto_images.py`：

```python
# 修改第 36 行
topic = "你的主题"  # 例如："北京美食探店"
```

### 方法 2: 创建自己的脚本

```python
from config import config
from mcp_client import MCPClient
from ai_generator import AIGenerator
from publisher import Publisher

# 初始化
mcp_client = MCPClient(config.mcp_server_url)
ai_generator = AIGenerator(
    api_key=config.gemini_api_key,
    base_url=config.gemini_base_url,
    model=config.gemini_model,
    proxy=config.gemini_proxy
)
publisher = Publisher(
    mcp_client,
    config.image_folder,
    config.local_image_folder,
    pixabay_api_key=config.pixabay_api_key
)

# 生成内容
topic = "你的主题"
result = ai_generator.generate_from_scratch(topic)

# 生成关键词并下载图片
keywords = ai_generator.generate_image_keywords(
    result["title"], 
    result["content"], 
    count=3
)
images = publisher.download_images_by_keywords(keywords, count=3)

# 发布
publisher.publish(
    title=result["title"],
    content=result["content"],
    images=images
)
```

## 🔧 常见问题

### Q1: Gemini API 报地区限制错误？

**解决方案：**
1. 确保代理软件正在运行
2. 切换代理节点到**美国**或**日本**
3. 验证代理：`python test_proxy_connection.py`（已删除，需要时重新创建）

### Q2: 图片下载失败？

**自动回退：**
程序会自动使用本地图片，不影响发布。

**手动解决：**
1. 检查 Pixabay API Key 是否有效
2. 检查网络连接
3. 手动准备图片放到 `E:\docker-xiaohongshu\images`

### Q3: 如何修改生成的内容风格？

编辑 `ai_generator.py` 中的 prompt：

```python

prompt = f"""
你是一个小红书博主，请根据主题创作一篇小红书笔记。

主题：{topic}

要求：
1. 标题要吸引眼球，多用emoji
2. 内容要口语化、接地气
3. 多用emoji增加趣味性
4. 适当使用网络流行语
5. 字数控制在200-500字
...

```

### Q4: 如何批量发布？

创建循环脚本：

```python
topics = [
    "上海咖啡店探店",
    "北京美食推荐",
    "杭州旅游攻略"
]

for topic in topics:
    result = ai_generator.generate_from_scratch(topic)
    keywords = ai_generator.generate_image_keywords(...)
    images = publisher.download_images_by_keywords(keywords)
    publisher.publish(...)
    
    import time
    time.sleep(60)  # 间隔 1 分钟
```
