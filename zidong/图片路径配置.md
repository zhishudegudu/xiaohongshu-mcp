# 图片路径配置说明

## 两种运行模式

### 模式 1: Docker 容器模式（推荐）

**适用场景**：MCP 服务器运行在 Docker 容器中

**配置方法**：
```bash
# Windows PowerShell
$env:USE_DOCKER="true"
python example_simple.py

# 或在代码中设置
import os
os.environ["USE_DOCKER"] = "true"
```

**路径映射**：
- 宿主机目录：`E:\xiao\xiaohongshu-mcp\docker\images\`
- 容器内路径：`/app/images/`
- Python 使用：`/app/images/`（自动配置）

**使用步骤**：
1. 将图片放到：`E:\xiao\xiaohongshu-mcp\docker\images\`
2. 设置环境变量：`$env:USE_DOCKER="true"`
3. 运行程序：`python example_simple.py`

---

### 模式 2: 本地模式（默认）

**适用场景**：MCP 服务器直接在本地运行（不使用 Docker）

**配置方法**：
```bash
# 默认就是本地模式，无需设置
python example_simple.py

# 或显式设置
$env:USE_DOCKER="false"
python example_simple.py
```

**路径配置**：
- 图片目录：`E:\新建文件夹\`
- Python 使用：`E:\新建文件夹\`（自动配置）

**使用步骤**：
1. 将图片放到：`E:\新建文件夹\`
2. 直接运行：`python example_simple.py`

## 快速配置

### 方法 1: 环境变量（推荐）

**Docker 模式**：
```powershell
$env:USE_DOCKER="true"
python example_simple.py
```

**本地模式**：
```powershell
$env:USE_DOCKER="false"
# 或不设置（默认就是 false）
python example_simple.py
```

### 方法 2: 修改 config.py

直接修改默认路径：
```python
def get_default_image_folder() -> str:
    if platform.system() == "Windows":
        # 改成你的路径
        return r"E:\你的图片文件夹"
```

### 方法 3: 代码中指定

在运行前设置：
```python
import os
os.environ["USE_DOCKER"] = "true"  # 或 "false"

# 然后正常运行
from config import config
print(config.image_folder)  # 查看使用的路径
```

## 常见问题

### Q1: 如何判断我应该用哪种模式？

**A**: 看你的 MCP 服务器怎么启动的：
- 使用 `docker-compose up` 或 `docker run` → **Docker 模式**
- 直接运行 Go 程序 `./app` → **本地模式**

### Q2: 我用 Docker，但图片在 `E:\新建文件夹\`，怎么办？

**A**: 修改 `docker-compose.yml`：
```yaml
volumes:
  - E:\新建文件夹:/app/images  # 改成你的路径
```

然后重启容器：
```bash
docker-compose down
docker-compose up -d
```

### Q3: 提示"图片文件不存在"？

**A**: 检查：
1. 确认运行模式（Docker 还是本地）
2. 确认图片在正确的目录
3. 运行 `python 验证配置.py` 查看当前配置

### Q4: 可以手动指定图片路径吗？

**A**: 可以！
```python
# Docker 模式
images = ["/app/images/1.jpg", "/app/images/2.jpg"]

# 本地模式
images = [r"E:\新建文件夹\1.jpg", r"E:\新建文件夹\2.jpg"]

publisher.publish(title, content, tags, images=images)
```

## 总结

| 模式 | 环境变量 | 图片位置 | Python 使用路径 |
|-----|---------|---------|----------------|
| **Docker** | `USE_DOCKER=true` | `E:\xiao\xiaohongshu-mcp\docker\images\` | `/app/images/` |
| **本地** | `USE_DOCKER=false`（默认） | `E:\新建文件夹\` | `E:\新建文件夹\` |

**推荐做法**：
1. 确定你的运行模式
2. 设置对应的环境变量
3. 将图片放到对应目录
4. 运行程序

**最简单的方式**：
```bash
# Docker 模式
$env:USE_DOCKER="true"
python example_simple.py

# 本地模式（默认）
python example_simple.py
```
