# 🚀 快速开始指南

## 📋 前提条件（必须满足）

在 Windows 上运行前，请确认：

- [x] **Python 3.8+** 已安装
- [x] **PyCharm** 已安装（或其他 IDE）
- [x] **小红书 MCP 服务器** 正在运行（Go 程序）
- [x] **Gemini API Key** 已获取

---

## ⚡ 三步快速配置

### 第 1 步：修改配置文件

打开 `config.py`，修改这 3 个地方：

```python
# 第 25 行：填入你的 Gemini API Key
gemini_api_key: str = "AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXX"  # ← 改这里

# 第 33 行：MCP 服务器地址
# 如果 Python 和 MCP 在同一台 Windows 机器上：
mcp_server_url: str = "http://127.0.0.1:18060/mcp"

# 如果 MCP 在其他机器上（比如 WSL）：
# mcp_server_url: str = "http://192.168.21.59:18060/mcp"

# 第 36 行：图片文件夹路径
image_folder: str = r"C:\Users\你的用户名\Pictures\xiaohongshu"  # ← 改这里
```

### 第 2 步：准备图片

1. 创建文件夹：`C:\Users\你的用户名\Pictures\xiaohongshu`
2. 放入至少 3 张图片（.jpg, .png, .jpeg）

### 第 3 步：安装依赖

在 PyCharm 终端中运行：

```bash
pip install -r requirements.txt
```

## 📍 关于 MCP 服务器地址

### 判断方法

**你的 MCP 服务器在哪里运行？**

1. **在同一台 Windows 机器上**
   ```python
   mcp_server_url = "http://127.0.0.1:18060/mcp"
   ```

2. **在 WSL (Windows 子系统) 中**
   ```python
   mcp_server_url = "http://localhost:18060/mcp"
   # 或
   mcp_server_url = "http://192.168.21.59:18060/mcp"
   ```

3. **在其他机器上**
   ```python
   mcp_server_url = "http://其他机器的IP:18060/mcp"
   ```

### 测试方法

在浏览器中访问：
```
http://127.0.0.1:18060
```

如果能看到页面（即使是错误页面），说明地址正确。

## ❌ 常见问题

### 问题 1：无法连接 MCP 服务器

**错误：**
```
❌ 连接失败: 无法连接到 MCP 服务器
```

**解决：**
1. 检查 MCP 服务器是否运行：`netstat -ano | findstr :18060`
2. 检查地址是否正确
3. 在浏览器中测试：`http://127.0.0.1:18060`

### 问题 2：图片文件夹不存在

**错误：**
```
❌ 文件夹不存在
```

**解决：**
1. 检查路径格式（Windows 路径）
2. 确保使用 `r` 前缀：`r"C:\Users\wu\Pictures\xiaohongshu"`
3. 或使用正斜杠：`"C:/Users/wu/Pictures/xiaohongshu"`

### 问题 3：API Key 错误

**错误：**
```
❌ Invalid API key
```

**解决：**
1. 访问 https://aistudio.google.com/app/apikey
2. 复制完整的 API Key
3. 粘贴到 `config.py` 第 25 行

### 问题 4：未登录小红书

**错误：**
```
登录状态: ❌ 未登录
```

**解决：**
需要先使用 MCP 客户端登录：
- 使用 Claude Desktop
- 或使用 MCP Inspector
- 调用 `get_login_qrcode` 工具
- 用小红书 App 扫码
