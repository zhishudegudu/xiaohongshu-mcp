## 代理问题修复

### 问题
代理配置导致本地 MCP 请求超时

### 解决方案
1. 在 `config.py` 中添加 `NO_PROXY` 环境变量（大写）
2. 在 `mcp_client.py` 中显式禁用所有 MCP 请求的代理

```python
# config.py
os.environ["no_proxy"] = "localhost,127.0.0.1,::1,0.0.0.0"
os.environ["NO_PROXY"] = "localhost,127.0.0.1,::1,0.0.0.0"

# mcp_client.py
proxies = {"http": None, "https": None}
response = requests.get(url, proxies=proxies, ...)
```
